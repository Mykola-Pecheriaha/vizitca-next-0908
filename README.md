
# Next.js + TypeScript + Turbopack + Tailwind CSS v4.0 + CSR

## Описание
- Next.js 15
- TypeScript
- Turbopack (dev)
- Tailwind CSS v4.0
- Yarn
- Только CSR (Client Side Rendering)

## Запуск

### Установка зависимостей
```
yarn install
```

### Запуск dev-сервера
```
yarn dev
```

### Сборка production
```
yarn build
```

### Запуск production-сервера
```
yarn start
```
